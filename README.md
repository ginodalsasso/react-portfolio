# Gino's Portfolio - React Web Application

## Overview
A modern, responsive portfolio website showcasing my transition from carpentry to web development. Features interactive 3D elements, bilingual support (EN/FR), and a comprehensive presentation of my skills and projects.

## Features
- **Interactive 3D Elements**: Three.js animations with rotating shapes and star fields
- **Bilingual Support**: Complete English/French language switching
- **Responsive Design**: Optimized for all devices
- **Project Showcase**: Interactive cards with GitHub integration
- **Contact Form**: Functional form with EmailJS and validation
- **Experience Timeline**: Dynamic vertical timeline
- **Smooth Animations**: Framer Motion scroll-triggered effects

## Technologies Used
- **Framework**: React
- **3D Graphics**: Three.js
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Form Handling**: EmailJS
- **Routing**: React Router DOM
- **Build Tool**: Vite

## Objectives
- Demonstrate technical proficiency in modern web development
- Showcase the unique transition from traditional craftsmanship to digital development
- Provide an engaging platform for potential employers and collaborators
